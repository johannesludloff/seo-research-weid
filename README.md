# SEO Research: Immobilienmakler Metropolregion Nürnberg (ER / N / FÜ)

Dieses Repository enthält eine Analyse der Wettbewerber für **WEID Immobilien** im Raum Erlangen, Nürnberg und Fürth.

## Wettbewerber-Analyse

### 1. WOGE Immobilien (Nürnberg)
*   **Website:** [woge-immobilien.de](https://www.woge-immobilien.de/)
*   **Fokus:** Verkauf & Vermietung von Wohnimmobilien (Haus, Wohnung, Grundstück).
*   **Seitenstruktur:**
    *   Startseite mit Fokus auf Auszeichnungen (Bellevue Best Property Agent 2026).
    *   Immobilienangebote (Kategorien: Kaufen, Mieten, Gewerbe).
    *   Referenzen & Teamvorstellung.
    *   Ratgeber-Sektion (SEO-Content).
*   **SEO-Keywords:** "Immobilienmakler Nürnberg", "Haus kaufen Nürnberg", "Wohnung vermieten Nürnberg".
*   **Eingesetzte Tools:**
    *   CRM: Wahrscheinlich onOffice oder FlowFact (Standard in der Branche).
    *   Tracking: Google Analytics, Facebook Pixel.
    *   Newsletter: Wahrscheinlich Mailchimp oder integriertes CRM-Tool.

### 2. Sparkassen-Immobilien (Erlangen/Nürnberg)
*   **Website:** [immobilien.sparkasse.de](https://immobilien.sparkasse.de/)
*   **Fokus:** Breite Marktabdeckung durch Banken-Netzwerk.
*   **Seitenstruktur:**
    *   Zentrales Suchportal für Immobilien.
    *   Regionale Unterseiten (z.B. Erlangen).
    *   Online-Preisfinder (Lead-Magnet).
*   **SEO-Keywords:** "Immobilien Erlangen", "Hausbewertung Erlangen", "Grundstück kaufen Nürnberg".
*   **Besonderheiten:** Extrem starke Domain-Authority durch die Sparkassen-Hauptdomain.
*   **Eingesetzte Tools:**
    *   S-ImmoPreisfinder (Eigengewächs/White-Label).
    *   Eigene Suchmaske mit Anbindung an LBS/Sparkassen-Bestand.

## Zusammenfassung & Empfehlungen für WEID Immobilien

1.  **Lead-Magnete:** Wettbewerber wie die Sparkasse nutzen Online-Bewertungstools. WEID sollte ebenfalls einen "Instant Valuation" Flow anbieten.
2.  **Regionale Keywords:** Fokus auf Long-Tail Keywords wie "Exklusive Wohnung Erlangen" oder "Haus verkaufen Erlangen Tennenlohe", um gegen die großen Portale zu bestehen.
3.  **Technischer Stack:** Die Nutzung von Propstack ist ein Vorteil gegenüber älteren Systemen (WOGE/Sparkasse). Das sollte im Frontend durch Schnelligkeit (Next.js) ausgespielt werden.
