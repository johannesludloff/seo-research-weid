## 2. Empfohlene Seitenstruktur

### Hauptseiten (Primäre Navigation)
```
/                           → Startseite (Hero + Trust Signals + CTA)
/verkaufen                  → Verkäufer-Landing (Lead-Magnet: Bewertung)
/kaufen                     → Käufer-Landing (Suchfilter + Objekte)
/immobilien                 → Objektübersicht (Propstack-Feed)
/wertermittlung             → Kostenlose Bewertung (Lead-Formular)
/ueber-uns                  → Team, Geschichte, Auszeichnungen
/kontakt                    → Kontaktformular + Standort
```

### Stadtteil-Seiten (Local SEO Booster)
```
/stadtteil/[name]           → Dynamische Seiten pro Stadtteil

  - Durchschnittspreise (m²)

  - Infrastruktur (Schulen, ÖPNV, Einkaufen)

```


## 3. Wettbewerber-Analyse: Was funktioniert?

### DAHLER Erlangen/Fürth
✅ **Stärken:**
- Premium-Positionierung ("hochwertige Immobilien", "Villen")
- Stadtteil-Content (Burgberg, Sieglitzhof, Alterlangen erwähnt)
- Marktdaten/Preisberichte als Lead-Magnet
- Testimonials ("Käufer nach 4 Wochen gefunden")

❌ **Schwächen:**
- Sehr allgemein, wenig tiefe lokale Inhalte
- Keine echten Blog-Artikel

### Grüske Immobilien
✅ **Stärken:**
- "Seit 1966" = Starkes Trust-Signal
- FAQ-Sektion (gut für SEO)
- Video-Content auf der Startseite
- Klare Leistungsübersicht

❌ **Schwächen:**
- Veraltetes Design
- Keine Stadtteil-Seiten
- Kein Blog/Ratgeber

### Immopartner / IMMOsmile
✅ **Stärken:**
- Umfangreiche Ratgeber-Artikel
- "7 Tipps zum Wohnungsverkauf" = gutes Content-Format
- Regionale Marktberichte

❌ **Schwächen:**
- Generische Designs
- Wenig differenzierend

## 5. Technische SEO-Empfehlungen

### On-Page Optimierung
- **Title Tags:** "Immobilienmakler Erlangen | WEID Immobilien" (max. 60 Zeichen)
- **Meta Descriptions:** Handlungsorientiert mit Call-to-Action
- **H1-Tags:** Exakt ein H1 pro Seite, Keyword enthalten
- **Schema Markup:** LocalBusiness, RealEstateAgent, FAQPage

### Core Web Vitals (Technischer Vorteil)
Da WEID mit Next.js gebaut wird, sollte die Seite bei den Core Web Vitals (LCP, FID, CLS) deutlich besser abschneiden als die veralteten Wettbewerber-Seiten. Das ist ein direkter Ranking-Faktor.

### Interne Verlinkung
- Jede Stadtteil-Seite verlinkt auf relevante Objekte aus Propstack
- Ratgeber-Artikel verlinken auf `/wertermittlung` als CTA
- Footer: Alle wichtigen Seiten verlinken
